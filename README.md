# Darwin World  
Simulation inspired by Charles Darwin theory of evolution. 
App allows user to configure various parameters e.g. map size, animal starting energy, energy lost daily,
cost of reproduction.  

Project realized as part of Object-Oriented Programing course at the Faculty of Computer Science at AGH University of Krakow.
